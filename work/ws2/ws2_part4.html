Now you can also draw the circle.The implementation was easy to extend
The second point of the circle is used for calculating the radius.
There circle approximate round with 11-triangle (suffient to make use seem like a cheap approximation of round) The worksheets proposed a quite complicated solution, and i chose a different approach (IMO simpler)
Implementation still use the same Object-style approach:
a) There is a list of completed shapes, each with a describtion of points + which drawmode to use
b) There is a preliminary object which will receive the new points the user clicks
c) Every time the preliminary object have enough points to complete the choosen shape it will add to the list of completed shapes
d) The render function draws both completed shapes and the preliminary object.
